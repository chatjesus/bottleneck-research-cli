# Code of Conduct

## Our Standard

This project is for building a professional, research-only open-source tool.
Participants are expected to communicate clearly, respectfully and in good faith.

Examples of expected behavior:

- focus on evidence and reproducible behavior
- respect financial-content boundaries
- disclose assumptions and uncertainty
- avoid personal attacks or inflammatory language
- avoid posting private data, credentials or personal portfolio information

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment or abusive language
- doxxing or publishing private information
- posting credentials, private endpoints or personal financial data
- pressuring contributors or users into trades
- using the project to promote guaranteed returns or personalized trading calls

## Enforcement

Maintainers may remove comments, close issues, reject pull requests or restrict
participation when behavior undermines the project boundary or community safety.
