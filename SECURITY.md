# Security Policy

## Supported Versions

The public repository currently supports the latest `main` branch.

## Reporting a Security Issue

Please do not open a public issue for:

- exposed credentials or tokens
- private data exposure
- subscriber data exposure
- vulnerabilities that could be used to access protected systems
- accidental publication of personal portfolio data

If you find a security issue, contact the project maintainers through the
website contact path at:

https://bottleneckresearch.com

Include:

- a short description of the issue
- affected command or file, if applicable
- reproduction steps
- whether any secret, private endpoint or personal data is involved

## Scope

This repository contains only the open-source CLI and public documentation. It
does not contain the private research pipeline, email system, subscriber
database, Cloudflare Worker configuration or paid/private data-source
credentials.

## Data Handling Boundary

Do not submit private portfolio screenshots, subscriber emails, API keys,
brokerage exports or paywalled research content in public issues or pull
requests.
